My Work

QuickDoc- Healthcare Appointment Platform

QuickDoc – Healthcare Appointment Platform is a full-stack web application built using React.js, Vite, Tailwind CSS, Node.js, Express.js, and MongoDB. It allows patients to search doctors by specialty, book appointments, and manage their medical profiles through a smooth and user-friendly interface. The platform follows a modular architecture with dedicated workflows for users, doctors, and admins, ensuring scalability and maintainability. Secure JWT-based authentication, role-based access control, real-time data handling, and Cloudinary integration for image uploads enhance both security and functionality.

AI Travel Planner

AI Travel Planner is an AI-powered web application built using React.js and Firebase that provides personalized trip planning experiences. It uses Google OAuth for secure user authentication and leverages the Google Gemini API to generate intelligent, customized travel itineraries. The application integrates third-party APIs like Pexels for rich destination visuals and LocationIQ for map-based exploration, enhancing user engagement and making trip suggestions more relevant and interactive.
